koolshare下Entware安装

July 24, 2019
Page content
      • Entware
      • ONMP

Entware 使用 opkg 包管理系统进行管理, 是一个适用于嵌入式系统的软件包库，迄今为止在官方的源上已经有超过 2000 个软件包。

近期为了搭建个人网盘在安装ONMP期间遇到了如下错误!

实际的关键错误代码

Unknown package 'sed'.
Collected errors:
* opkg_install_cmd: Cannot install package sed.
Unknown package 'nginx-extras'.
Collected errors:
* opkg_install_cmd: Cannot install package nginx-extras.

可知是软件包中未能找到相关的包,经过一段时间的搜索找到了如下靠谱的解决方案。

Entware

wget -O - http://bin.entware.net/x64-k3.2/installer/generic.sh | /bin/sh

ONMP

sh -c "$(curl -kfsSl https://pyocean.com/data/bashs/onmp.sh)"

在运行如上面命令后，ONMP被成功的安装。
