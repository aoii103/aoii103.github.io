vue路径栏中?号与#号问题

October 07, 2019
Page content
  • 路径栏#号
  • 路径栏问号导致刷新

在vue项目开发过程中，发现了这两个问题。一个不美观，另一个会导致界面刷新，在这里记录一下。

路径栏#号

通过修改总路由代码增加mode:'history'以去除地址栏#号。

export default new Router({
    mode:'history',
    routes:x_router
})

路径栏问号导致刷新

通过对el-form增加@submit.native.prevent防止在点击search按钮后地址栏出现额外?号并刷新界面。

<el-form @submit.native.prevent>
   <el-form-item>
       <el-input v-model="data"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button @click="search">查询</el-button>
  </el-form-item>
</el-form>
