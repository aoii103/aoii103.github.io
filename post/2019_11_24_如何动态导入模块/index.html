如何动态导入模块之imp

November 24, 2019
Page content

在我定制poc漏扫的开发过程成，需要动态从数据导出并添加某些模块，记录下这个过程

这段代码适用于python3环境。

import imp

code = """
def send(data):
    return data+'!'
"""
module = imp.new_module('name')
exec(code,module.__dict__)
module.send('test')
'test!'
