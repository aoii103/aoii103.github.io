Shadowsock作为普遍用户第一的翻墙选择，已经变得泛滥。高识别率，加密方式存在已知缺陷等等都让它荣光不再。而这种时候，V2ray站了出来。让我们一起来看看吧!

服务端安装

首先我们在我们准备好的ces上运行如下命令

bash <(curl -s -L https://git.io/v2ray.sh)

首先我们输入1进入安装选项。

然后我们选择我们所喜欢的协议，默认为第一个。

接着我们选择需要使用的端口及考虑下要不要顺带开启shadowsock服务。

最后我们按任意键开始自动安装部署，等跑完之后服务就安装完毕了。

MAC客户端

这边我们使用V2RayX 作为GUI客户端，通过如下命令来一键安装。

brew cask install v2rayx

然后我们打开客户端的configure选项，填入相应的参数即可。

IOS客户端

potatso Lite

shadowrocket

这边我们推荐使用potatso Lite作为客户端，而曾经风靡一时的小火箭shadowrocket已经在国内被下架了。除非您有国外的appid，即可付费下载(现已收费)
