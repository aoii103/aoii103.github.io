以下未注明安装方式的在mac下均可通过 brew install xxx 直接安装

0x01 翻译

fy 如果你只是想简单得查查单词，它或许会是你最好的选择。这是一款中英文互译工具，自动判断语言并拼读单词，通过 npm install fanyi -g 安装。

trans 这是一款多语言互译工具，可整句翻译、调用多种引擎、功能齐全且跨平台。Mac下可通过brew install translate-shell安装

0x02 命令

tldr 当你不了解某个命令的用法，并不想查看--help的时候，它的作用就显而易见了。

cht.sh 或许你渴望更全面的语法案例，那就选它。个人认为的唯一缺点就是结果高亮效果较差。 访问 https://github.com/chubin/cheat.sh#installation 以安装

0x03 搜索

googler 可直接返回谷歌搜索结果，免去了打开浏览器的过程(查看详情的时候还是要的)

fd 相比前一位，它更擅长的是文件搜索，支持正则、指定文件类型及隐藏目录查找。

0x04 预览

bat 都看到了这里，那想必大家一定熟悉cat吧，更有甚者也了解 ccat 等等。它们是不是都很伤眼睛，是的话那就下载它吧。

imgcat 查找到了图片，又想快速得查看，那就用它了。通过 pip install imgcat 进行安装。
